Fire Detection using Neural Networks

Project Organization

    ├── README.md          <- Project documentation
    │
    ├── models             <- Models included in the project
    │   ├── models.py
    │   
    ├── notebooks          <- Jupyter notebooks of examples of how to use the materials.
    │
